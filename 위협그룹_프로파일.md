# 💀 그룹 2 — GORGON RaaS

## 1. 개요

| 항목 | 내용 |
|---|---|
| 그룹명 | GORGON |
| 분류 | 서비스형 랜섬웨어 (RaaS) 운영 조직 (가상) |
| 실제 TTPs 기반 | Medusa RaaS (Spearwing 운영) |
| 주요 목적 | 데이터 암호화 + 이중 협박 (Double Extortion) |
| IRON VEIL과의 관계 | 어필리에이트 계약 — IRON VEIL이 초기 접근권 제공, GORGON이 랜섬웨어 배포 |
| 역할 분담 | IRON VEIL: 침투·정보탈취 / GORGON: 교란·랜섬웨어 배포 |

## 2. 주요 TTPs

| 기법 | MITRE ID | 설명 | 시나리오 적용 자산 |
|---|---|---|---|
| 초기 접근권 구매 | T1078 | IRON VEIL로부터 공공기관 INT 접근권 이전 | 공공기관 INT |
| 자격증명 덤프 | T1003.001 | Mimikatz로 추가 계정 확보 | 업무용 PC |
| 네트워크 공유 탐색 | T1135 | 암호화 대상 파일 범위 파악 | INT 파일서버, DB |
| 백업 삭제 | T1490 | VSS(볼륨 섀도 복사본) 삭제로 복구 차단 | Windows 서버·PC |
| 데이터 암호화 | T1486 | GORGON 랜섬웨어 배포 및 파일 암호화 | 공공기관 INT 서버·PC |
| 이중 협박 | T1657 | 암호화 전 데이터 유출 후 공개 협박 | DB 덤프 데이터 |

## 3. IRON VEIL ↔ GORGON 협력 구조

```
IRON VEIL                          GORGON
────────────────────────────────────────────────────
1. 공공기관 INT 침투 완료
2. DB 덤프·기밀 탈취 완료
3. 군 VPN 크리덴셜 확보
4. ──────────────────────────────→ 접근권 + DB 덤프 전달
5. 군 DMZ → C4I 침투 계속 진행     5. 공공기관 INT에 랜섬웨어 배포
6. C4I 데이터 조작                 6. 몸값 협상 및 이중 협박
────────────────────────────────────────────────────
→ IRON VEIL은 교란 연막 속에서 조용히 군 C4I 공격 완료
```

## 🏢 공공기관 DMZ

### 1. 외부 포털 서버 (React + FastAPI)
| 항목 | 내용 |
|---|---|
| 공격 주체 | IRON VEIL |
| 공격 방법 | API 인증 미흡 취약점 → 웹셸 이식 또는 SSRF로 내부 정보 탈취 |
| 사용 도구 | Burp Suite, 커스텀 웹셸(Python/ASPX) |
| 검증 결과 | ✅ FastAPI 엔드포인트 인증 미흡 → 관리자 API 직접 호출 가능. 웹셸 업로드는 파일 업로드 기능 없으면 제한적 → **SSRF 또는 SQLi 경로가 더 자연스러움** |
| 구현 포인트 | FastAPI 라우터에 인증 미적용 엔드포인트 하나 삽입, 또는 SQL Injection으로 데이터 탈취 후 DMZ→INT 피벗 |

### 2. 메일 포털 게이트 (Nginx Reverse Proxy)
| 항목 | 내용 |
|---|---|
| 공격 주체 | IRON VEIL |
| 공격 방법 | 프록시 헤더 조작으로 INT 웹메일 직접 접근, 세션 탈취 |
| 검증 결과 | ✅ Nginx `proxy_pass` 설정 오류로 `/internal/` 경로 노출 가능. 현실적이고 구현 용이 |
| 구현 포인트 | Nginx 설정에서 내부 경로 일부를 접근제어 없이 프록시하도록 의도적 오설정 |

### 3. 민원 접수 서버 (React + FastAPI + MinIO)
| 항목 | 내용 |
|---|---|
| 공격 주체 | IRON VEIL |
| 공격 방법 | 파일 업로드 검증 미흡 → 악성 파일 업로드 → 민원처리서버 파이프라인에서 실행 |

| 구현 포인트 | FastAPI 업로드 엔드포인트에서 확장자만 검사(콘텐츠 미검사), MinIO presigned URL 권한 오설정으로 다른 민원 파일 접근 가능 |
