제3장 군 네트워크 접속 절차

제6조 (군 VPN 접속 체계)
행정안전부 협력 담당자가 국방부 시스템에 접속할 때는 군 전용 VPN을
사용해야 한다.

1. VPN 접속 주소: vpn.mnd.valdoria.mil (211.57.64.11)
2. VPN 클라이언트: 국방부 전용 VPN 클라이언트 (FortiClient 호환)
3. 접속 가능 시간: 평일 08:00 ~ 20:00 (긴급 시 24시간)
4. 접속 허용 IP: 행정안전부 내부망 (192.168.100.0/24)

제7조 (VPN 계정 발급 및 관리)
1. 계정 형식: GOV + 사번 (예: GOV20190847)
   - GOV: 정부기관(Government) 외부 접속자 표시
   - 사번: 행정안전부 인사관리 시스템 사번

2. 초기 비밀번호: 사번 + 생년월일 6자리 (YYMMDD → 아닌 YYYYMMDD에서 앞 2자리 제외하지 않고 6자리)
   - 형식: [사번8자리][생년월일6자리] (총 14자리)
   - 예시: 사번 20190847, 생년월일 1989-03-12 → 비밀번호: 20190847890312
   - 최초 접속 시 반드시 비밀번호 변경 필요

3. 계정 유효기간: 발급일로부터 6개월 (갱신 가능)
4. 비밀번호 변경 주기: 60일
5. 5회 인증 실패 시 계정 잠금 → 국방부 민관협력과에 해제 요청

제8조 (접속 절차)
1. VPN 클라이언트 실행
2. 서버 주소 입력: vpn.mnd.valdoria.mil
3. 계정 입력: GOV + 사번
4. 비밀번호 입력
5. 접속 승인 후 군 내부 시스템 접근 가능

제9조 (접속 제한 사항)
1. 동시 접속 1세션만 허용
2. 30분 이상 미사용 시 자동 연결 해제
3. 파일 전송은 승인된 경로를 통해서만 가능
4. 접속 로그는 12개월간 보관 및 정기 감사 대상

제4장 협력 업무 수행

제10조 (재난 대응 군 지원 요청)
1. 대규모 재난 발생 시 행정안전부가 국방부에 군 지원 요청
2. 요청 절차: 재난대응과 → 장관 승인 → 국방부 민관협력과
3. 지원 유형: 인력 투입, 장비 지원, 헬기 운용 등

제11조 (사이버 위협 정보 공유)
1. 양 기관 간 사이버 위협 지표(IOC) 실시간 공유
2. 공유 플랫폼: 군 CERT ↔ 행안부 보안정책과
3. 월 1회 합동 위협 분석 회의 개최

제12조 (인사 교류)
1. 연 2회 단기 인사 교류 (2주간)
2. 교류 대상: 사이버 보안, 재난 대응 분야 실무진
3. 교류 인원: 기관당 2~3명
