## 1. 개요

### 1.1 자산 목적

웹메일 서버는 발도리아 국방부(MND) 군 내부 직원을 위한 이메일 서비스를 제공하는 서버이다. Postfix(MTA), Dovecot(IMAP), Roundcube(WebUI)의 3개 오픈소스 구성 요소를 조합하여 메일 송수신, 저장 및 웹 기반 열람 기능을 제공한다.

주요 기능:

- **메일 송수신**: Postfix SMTP를 통한 군 내부 메일 라우팅 및 배달
- **메일 저장/열람**: Dovecot IMAP을 통한 메일함 접근
- **웹 인터페이스**: Roundcube를 통한 브라우저 기반 메일 읽기/쓰기/관리
- **AD 인증 연동**: Active Directory(192.168.92.230)를 통한 통합 계정 인증

### 1.2 훈련에서의 역할 (STEP 4-4)

이 자산은 레드팀의 **CVE-2025-49113 RCE 공격 대상** 및 **내부 피싱 메일 발송 거점**으로 설계되었다. 공격자는 Roundcube의 플러그인 처리 과정에서 발생하는 RCE 취약점(CVE-2025-49113)을 활용하여 웹셸을 업로드하고, 군 내부 직원에게 추가 피싱 메일을 발송한다.

## 5. CVE-2025-49113: Roundcube RCE (훈련용 가상 CVE)

### 5.1 취약점 개요

> **참고: CVE-2025-49113은 훈련 전용 가상 CVE이다. 실제 존재하지 않는 취약점이며, 현실적인 RCE 시나리오로 설계하였다.**

| 항목 | 내용 |
|------|------|
| CVE ID | CVE-2025-49113 (가상) |
| 영향 소프트웨어 | Roundcube Webmail 1.6.0 ~ 1.6.6 |
| 유형 | Remote Code Execution (RCE) |
| CVSS | 9.8 (Critical) |
| 공격 벡터 | 네트워크, 인증 불필요 |
| 근본 원인 | 플러그인 설정 파일 처리 과정에서 PHP 역직렬화(Deserialization) 취약점 |

### 5.2 기술적 상세

Roundcube 1.6.6 이하 버전의 플러그인 핸들러(`program/lib/Roundcube/rcube_plugin_api.php`)에서 플러그인 메타데이터를 처리할 때, HTTP 요청으로 전달된 직렬화된 PHP 객체를 `unserialize()` 함수로 역직렬화한다. 이 과정에서 공격자가 조작한 직렬화 데이터를 통해 임의 코드 실행이 가능하다.

**취약 코드 (의도적 삽입):**

```php
<?php
// program/lib/Roundcube/rcube_plugin_api.php
// ⚠ CVE-2025-49113: 플러그인 메타데이터 역직렬화 취약점

class rcube_plugin_api {

    // ... 기존 코드 ...

    /**
     * 플러그인 설정 로드 — 외부 플러그인 저장소에서 메타데이터 동기화 시 호출
     * ⚠ 취약: 사용자 입력을 직접 역직렬화
     */
    public function load_plugin_metadata($plugin_name) {
        // HTTP 요청에서 플러그인 메타데이터 수신
        $raw_metadata = rcube_utils::get_input_value('_plugin_meta', rcube_utils::INPUT_POST);

        if ($raw_metadata) {
            // ⚠ 취약: unserialize()에 사용자 입력 직접 전달
            // 정상적으로는 json_decode()를 사용해야 함
            $metadata = unserialize(base64_decode($raw_metadata));

            if (is_array($metadata) && isset($metadata['name'])) {
                $this->plugin_metadata[$plugin_name] = $metadata;
                return true;
            }
        }
        return false;
    }

    /**
     * 플러그인 관리 인터페이스 — 설치된 플러그인 목록 반환
     * 이 엔드포인트는 인증 없이 접근 가능 (⚠ 취약)
     */
    public function api_list_plugins() {
        // _plugin_meta 파라미터가 있으면 메타데이터 동기화 시도
        if (rcube_utils::get_input_value('_plugin_meta', rcube_utils::INPUT_POST)) {
            $this->load_plugin_metadata('__sync__');
        }

        $plugins = array_keys($this->plugins);
        header('Content-Type: application/json');
        echo json_encode(['plugins' => $plugins]);
        exit;
    }
}
```
