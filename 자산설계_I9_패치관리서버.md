## 1. 개요

### 1.1 자산 목적

패치 관리 서버는 군 내부 업무용 PC(192.168.110.31~35) 및 관리자 PC(192.168.110.40)에 보안 패치와 소프트웨어 업데이트를 배포하는 중앙 업데이트 서버이다. 전용 서브넷(192.168.120.0/24)에 배치되어 VPN 접속 사용자(172.20.100.x)가 원격으로 업데이트를 관리할 수 있도록 설계되었다.

### 1.2 훈련에서의 역할 (STEP 4-2) -- 핵심 공급망 공격 벡터

> **이 자산은 전체 군 INT 공격 체인의 핵심 진입점이다.**

공격자는 VPN 접속 후 이 서버에만 접근 가능하며, 약한 관리자 비밀번호를 브루트포스로 탈취한 후 정상 업데이트 파일을 악성 페이로드(Havoc C2 Demon)로 교체한다. 군 업무용 PC의 자동 업데이트 메커니즘이 악성 파일을 다운로드/실행하면서 내부망 전체가 감염된다.

### 1.3 공격 체인에서의 위치

```

    │
    │  OPNSense-6 규칙: 172.20.100.0/24 → 192.168.120.0/24 ONLY
    │  (192.168.110.0/24에는 직접 접근 불가)
    │
    ▼
[★ STEP 4-2] 패치 관리 서버 (192.168.120.10)
    │
    │  ① 관리자 패널 발견: http://192.168.120.10/admin/
    │  ② 브루트포스: admin / admin123
    │  ③ 로그인 → manifest.json 확인
    │  ④ SecurityPatch_KB2024001.exe 다운로드 (정상 파일 백업)
    │  ⑤ Havoc Demon 페이로드로 교체 업로드
    │  ⑥ manifest.json 해시 업데이트
    │
    ▼
```


### 6.3 auth.php

```php
<?php
/**
 * 인증 함수
 */

require_once __DIR__ . '/config.php';

session_start();

function authenticate($username, $password) {
    // ⚠ 취약: 단순 문자열 비교, 계정 잠금 없음, 지연 없음
    if ($username === ADMIN_USERNAME && $password === ADMIN_PASSWORD) {
        $_SESSION['authenticated'] = true;
        $_SESSION['username'] = $username;
        $_SESSION['login_time'] = time();

        // 로그인 기록
        $log_entry = date('Y-m-d H:i:s') . " LOGIN_SUCCESS {$username} from {$_SERVER['REMOTE_ADDR']}\n";
        file_put_contents(DOWNLOAD_LOG, $log_entry, FILE_APPEND);

        return true;
    }

    // 로그인 실패 기록 (⚠ 계정 잠금 없음 — 브루트포스 가능)
    $log_entry = date('Y-m-d H:i:s') . " LOGIN_FAILED {$username} from {$_SERVER['REMOTE_ADDR']}\n";
    file_put_contents(DOWNLOAD_LOG, $log_entry, FILE_APPEND);

    return false;
}

function check_auth() {
    if (!isset($_SESSION['authenticated']) || !$_SESSION['authenticated']) {
        header('Location: /admin/login.php');
        exit;
    }

    // 세션 타임아웃 확인
    if (time() - $_SESSION['login_time'] > SESSION_TIMEOUT) {
        session_destroy();
        header('Location: /admin/login.php?timeout=1');
        exit;
    }
}
```

## 7. Nginx 설정

```nginx
# /etc/nginx/sites-available/update-server
server {
    listen 80;
    server_name update.mnd.local 192.168.120.10;

    root /var/www/update-server;
    index index.html;

    # 업데이트 파일 서빙 (정적)
    location /updates/ {
        alias /var/www/update-server/updates/;
        autoindex off;

        # 다운로드 로그 (Nginx access log로 추적)
        access_log /var/log/nginx/update-download.log;
    }

    # 관리자 패널 (PHP)
    location /admin/ {
        alias /var/www/update-server/admin/;
```

## 8. Windows PC 업데이트 체커 스크립트

이 스크립트는 I10 업무용 PC 및 I11 관리자 PC에 Scheduled Task로 설치된다.

### 8.1 update-checker.ps1

```powershell
<#
.SYNOPSIS
    군 패치 관리 서버 자동 업데이트 클라이언트

.DESCRIPTION
    주기적으로 패치 관리 서버의 manifest.json을 확인하고,
    새로운 업데이트가 있으면 자동으로 다운로드 및 설치한다.

    ⚠ 훈련용: 의도적으로 보안 검증이 미흡하게 설계됨
    - HTTPS 미사용 (HTTP 평문)
    - 코드 서명 미검증
    - 관리자 권한으로 자동 실행
    - manifest 해시만 확인 (공격자가 둘 다 변경 가능)

.NOTES
    자산: I10 업무용 PC 1~5, I11 관리자 PC
    설치 위치: C:\ProgramData\MND\UpdateService\
    Scheduled Task: MND_UpdateChecker (30분 간격)
#>

# === 설정 ===
$UpdateServerUrl = "http://update.mnd.local"
$ManifestUrl = "$UpdateServerUrl/updates/manifest.json"
$LocalUpdateDir = "C:\ProgramData\MND\UpdateService"
$InstalledVersionsFile = "$LocalUpdateDir\installed_versions.json"
$LogFile = "$LocalUpdateDir\update.log"
$TempDownloadDir = "$LocalUpdateDir\temp"

# === 로깅 함수 ===
function Write-UpdateLog {
    param([string]$Message, [string]$Level = "INFO")
    $timestamp = Get-Date -Format "yyyy-MM-dd HH:mm:ss"
    $entry = "[$timestamp] [$Level] $Message"
    Add-Content -Path $LogFile -Value $entry
    if ($Level -eq "ERROR") {
        Write-Host $entry -ForegroundColor Red
    } else {
        Write-Host $entry
    }
}

# === 초기화 ===
function Initialize-UpdateService {
    if (-not (Test-Path $LocalUpdateDir)) {
        New-Item -ItemType Directory -Path $LocalUpdateDir -Force | Out-Null
    }
    if (-not (Test-Path $TempDownloadDir)) {
        New-Item -ItemType Directory -Path $TempDownloadDir -Force | Out-Null
    }
    if (-not (Test-Path $InstalledVersionsFile)) {
        '{}' | Out-File -FilePath $InstalledVersionsFile -Encoding UTF8
    }
}

# === 설치된 버전 관리 ===
function Get-InstalledVersions {
    try {
        $content = Get-Content -Path $InstalledVersionsFile -Raw
        return $content | ConvertFrom-Json
    } catch {
        return @{}
    }
}

function Set-InstalledVersion {
    param([string]$UpdateId, [string]$Version)
    $versions = Get-InstalledVersions
    $versions | Add-Member -NotePropertyName $UpdateId -NotePropertyValue $Version -Force
    $versions | ConvertTo-Json | Out-File -FilePath $InstalledVersionsFile -Encoding UTF8
}

# === 해시 검증 ===
function Get-FileHashSHA256 {
    param([string]$FilePath)
    $hash = Get-FileHash -Path $FilePath -Algorithm SHA256
    return $hash.Hash.ToLower()
}

# === 매니페스트 다운로드 ===
function Get-UpdateManifest {
    Write-UpdateLog "매니페스트 확인: $ManifestUrl"

    try {
        # ⚠ 취약: HTTP 평문 통신 — MITM 가능
        $response = Invoke-WebRequest -Uri $ManifestUrl -UseBasicParsing -TimeoutSec 30
        $manifest = $response.Content | ConvertFrom-Json
        Write-UpdateLog "매니페스트 버전: $($manifest.version), 업데이트 수: $($manifest.updates.Count)"
        return $manifest
    } catch {
        Write-UpdateLog "매니페스트 다운로드 실패: $_" -Level "ERROR"
        return $null
    }
}

# === 업데이트 다운로드 ===
function Download-Update {
    param([PSObject]$Update)

    $downloadUrl = "$UpdateServerUrl/updates/$($Update.filename)"
    $tempPath = "$TempDownloadDir\$($Update.filename)"

    Write-UpdateLog "다운로드 시작: $($Update.filename) ($($Update.version))"

    try {
        # ⚠ 취약: HTTP 평문 다운로드
        Invoke-WebRequest -Uri $downloadUrl -OutFile $tempPath -UseBasicParsing -TimeoutSec 120

        # 해시 확인 (⚠ manifest의 해시와 비교 — 공격자가 둘 다 변경하면 무력화)
        $fileHash = Get-FileHashSHA256 -FilePath $tempPath

        if ($fileHash -eq $Update.sha256) {
            Write-UpdateLog "해시 검증 성공: $($Update.filename)"
            return $tempPath
        } else {
            Write-UpdateLog "해시 불일치! 예상: $($Update.sha256), 실제: $fileHash" -Level "ERROR"
            Remove-Item -Path $tempPath -Force -ErrorAction SilentlyContinue
            return $null
        }
    } catch {
        Write-UpdateLog "다운로드 실패: $_" -Level "ERROR"
        return $null
    }
}

# === 업데이트 실행 ===
function Install-Update {
    param([string]$FilePath, [PSObject]$Update)

    Write-UpdateLog "업데이트 설치 시작: $($Update.name) ($($Update.version))"

    $extension = [System.IO.Path]::GetExtension($FilePath).ToLower()

    try {
        switch ($extension) {
            ".exe" {
                # ⚠ 취약: 코드 서명 미검증 — 임의 exe 실행
                # ⚠ 취약: 관리자 권한으로 자동 실행
                # ⚠ 취약: 사용자 확인 없음
                Write-UpdateLog "EXE 실행: $FilePath"
                $process = Start-Process -FilePath $FilePath -ArgumentList "/silent /norestart" `
                    -Wait -PassThru -NoNewWindow
                Write-UpdateLog "설치 완료 (종료 코드: $($process.ExitCode))"
            }
            ".msi" {
                Write-UpdateLog "MSI 설치: $FilePath"
                $process = Start-Process -FilePath "msiexec.exe" `
                    -ArgumentList "/i `"$FilePath`" /quiet /norestart" `
                    -Wait -PassThru -NoNewWindow
                Write-UpdateLog "설치 완료 (종료 코드: $($process.ExitCode))"
            }
            ".msp" {
                Write-UpdateLog "MSP 패치: $FilePath"
                $process = Start-Process -FilePath "msiexec.exe" `
                    -ArgumentList "/p `"$FilePath`" /quiet /norestart" `
                    -Wait -PassThru -NoNewWindow
                Write-UpdateLog "설치 완료 (종료 코드: $($process.ExitCode))"
            }
            default {
                Write-UpdateLog "지원하지 않는 형식: $extension" -Level "ERROR"
                return $false
            }
        }

        # 설치 버전 기록
        Set-InstalledVersion -UpdateId $Update.id -Version $Update.version

        # 임시 파일 정리
        Remove-Item -Path $FilePath -Force -ErrorAction SilentlyContinue

        return $true
    } catch {
        Write-UpdateLog "설치 실패: $_" -Level "ERROR"
        return $false
    }
}

# === 메인 실행 ===
function Start-UpdateCheck {
    Write-UpdateLog "========== 업데이트 확인 시작 =========="

    Initialize-UpdateService

    # 매니페스트 다운로드
    $manifest = Get-UpdateManifest
    if (-not $manifest) {
        Write-UpdateLog "매니페스트를 가져올 수 없습니다. 종료." -Level "ERROR"
        return
    }

    # 설치된 버전 확인
    $installedVersions = Get-InstalledVersions

    $updatesApplied = 0

    foreach ($update in $manifest.updates) {
        # 자동 설치 대상만 처리
        if (-not $update.auto_install) {
            Write-UpdateLog "건너뜀 (수동 설치): $($update.name)"
            continue
        }

        # 이미 설치된 버전 확인
        $installedVersion = $installedVersions.PSObject.Properties[$update.id]
        if ($installedVersion -and $installedVersion.Value -eq $update.version) {
            Write-UpdateLog "최신 버전 설치됨: $($update.name) v$($update.version)"
            continue
        }

        Write-UpdateLog "새 업데이트 발견: $($update.name) v$($update.version) [$($update.severity)]"

        # 다운로드
        $downloadedFile = Download-Update -Update $update
        if (-not $downloadedFile) {
            Write-UpdateLog "$($update.name) 다운로드 실패, 건너뜀" -Level "ERROR"
            continue
        }

        # 설치
        $success = Install-Update -FilePath $downloadedFile -Update $update
        if ($success) {
            $updatesApplied++
            Write-UpdateLog "$($update.name) 설치 성공"
        }
    }

    Write-UpdateLog "========== 업데이트 확인 완료 (적용: $updatesApplied 건) =========="
}

# 실행
Start-UpdateCheck
```

```powershell
    Add-Content -Path $hostsFile -Value $hostsEntry
}

    -Execute "powershell.exe" `
    -Argument "-ExecutionPolicy Bypass -File `"$ScriptPath`"" `
    -WorkingDirectory $ServiceDir

$Trigger = New-ScheduledTaskTrigger `
    -RepetitionInterval (New-TimeSpan -Minutes 30) `
    -RepetitionDuration (New-TimeSpan -Days 365) `
    -At (Get-Date).Date

    -AllowStartIfOnBatteries `
    -DontStopIfGoingOnBatteries `
    -StartWhenAvailable `
    -RunOnlyIfNetworkAvailable

$Principal = New-ScheduledTaskPrincipal `
    -UserId "SYSTEM" `
    -LogonType ServiceAccount `
    -RunLevel Highest

Unregister-ScheduledTask -TaskName $TaskName -Confirm:$false -ErrorAction SilentlyContinue

Register-ScheduledTask `
    -TaskName $TaskName `
    -Action $Action `
    -Trigger $Trigger `
    -Settings $Settings `
    -Principal $Principal `
    -Description "국방부 패치 관리 서버에서 보안 업데이트를 자동 확인하고 설치합니다."
```
