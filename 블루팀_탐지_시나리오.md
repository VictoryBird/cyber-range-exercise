# 🔵 블루팀 탐지 시나리오
> 본 문서는 레드팀 공격 완료 후 블루팀 사후 로그 분석 훈련용입니다.
> 분석 방향: 공격 타임라인 재구성 → 침투 경로 특정 → 피해 범위 확정

---

# 분석 원칙

- **로그만 믿을 것**: 시스템 현재 상태는 이미 조작됐을 수 있음
- **타임라인 우선**: 개별 이벤트가 아닌 흐름으로 읽어야 공격 체인이 보임
- **존 경계 로그 필수**: OPNSense 방화벽 로그가 횡이동 경로의 결정적 증거
- **블루팀 목표**: 침투 경로 재구성, 피해 자산 목록, 최초 침투 시점 특정

## 1-1. 외부 포털 서버 이상 징후

**분석 대상 로그**
- `/var/log/nginx/access.log`
- FastAPI 애플리케이션 로그
- PostgreSQL 쿼리 로그

**탐지 시그니처**

```bash
# [Nginx] 비정상 API 스캔 패턴
grep -E '(GET|POST).*/api/(admin|internal|v1/users|docs|openapi)' access.log \
  | awk '{print $1,$7,$9}' | sort | uniq -c | sort -rn
# 기준: 동일 IP에서 5분 내 10개 이상 상이한 API 경로 요청

# [Nginx] SQL Injection 시그니처
grep -E "(UNION|SELECT|INSERT|DROP|--|\x27|\x22)" access.log

# [FastAPI] 인증 없는 관리자 엔드포인트 접근 (HTTP 200 응답)
grep '/api/v1/admin' access.log | grep ' 200 '
```

**주요 탐지 지표 (IoC)**
| 지표 | 설명 |
|---|---|
| 단일 IP에서 5분 내 /docs, /openapi.json 접근 | API 구조 정찰 |
| `/api/v1/admin` 200 응답 | 인증 없는 관리자 API 접근 성공 |
| SQL 특수문자 포함 요청 | SQLi 시도 |

## 2-5. 업무용 PC — 자격증명 덤프 및 횡이동

**분석 대상 로그**: Windows Security 이벤트, Sysmon 로그

**탐지 시그니처**

```powershell
# [Windows Security] LSASS 프로세스 접근 (Event ID 4656)
Get-WinEvent -LogName Security | Where-Object {
  $_.Id -eq 4656 -and $_.Message -match "lsass.exe"
} | Select-Object TimeCreated, Message

# [Sysmon] Mimikatz 관련 프로세스 생성 (Event ID 1)
Get-WinEvent -LogName "Microsoft-Windows-Sysmon/Operational" | Where-Object {
  $_.Id -eq 1 -and ($_.Message -match "mimikatz" -or $_.Message -match "sekurlsa")
}

# [Sysmon] PowerShell 인코딩 실행
Get-WinEvent -LogName "Microsoft-Windows-Sysmon/Operational" | Where-Object {
  $_.Id -eq 1 -and $_.Message -match "powershell.*-enc"
}

# [Windows Security] Pass-the-Hash 탐지 (Event ID 4624, NTLM, Logon Type 3)
Get-WinEvent -LogName Security | Where-Object {
  $_.Id -eq 4624 -and $_.Message -match "NtLmSsp" -and $_.Message -match "LogonType.*3"
}
```

**타임라인 재구성 포인트**
```
피싱 메일 수신 (Postfix)
  → LNK 실행 (Sysmon Event 1)
  → Mimikatz 실행 (Event 4656)
  → 횡이동 (Event 4624, NTLM)
  → 관리자 PC PsExec (Sysmon Network Event 3)
```

## 6-1. 랜섬웨어 초기 탐지

**분석 대상 로그**: Windows Security 이벤트, Sysmon 로그

**탐지 시그니처**

```powershell
# [Sysmon] VSS 삭제 명령 (Event ID 1)
Get-WinEvent -LogName "Microsoft-Windows-Sysmon/Operational" | Where-Object {
  $_.Id -eq 1 -and (
    $_.Message -match "vssadmin.*delete" -or
    $_.Message -match "wmic.*shadowcopy.*delete"
  )
}

# [Windows Security] 짧은 시간 내 대량 파일 Write (Event ID 4663)
Get-WinEvent -LogName Security | Where-Object {
  $_.Id -eq 4663 -and $_.Message -match "WriteData"
} | Group-Object { $_.TimeCreated.Minute } | Where-Object { $_.Count -gt 100 }

# [Sysmon] 랜섬 노트 파일 생성 (Event ID 11)
Get-WinEvent -LogName "Microsoft-Windows-Sysmon/Operational" | Where-Object {
  $_.Id -eq 11 -and $_.Message -match "README_GORGON"
}
```
