## 1. 개요

### 1.1 자산 목적

D3 자료교환체계는 군 DMZ와 INT 간의 파일 교환을 위한 Nextcloud 기반 시스템이다. 군 내부 직원과 외부 협력기관 간 문서를 안전하게 공유하는 플랫폼으로 운영된다.

주요 기능:
- **파일 업로드/다운로드**: 브라우저 또는 WebDAV 클라이언트
- **공유 폴더**: 부서별/프로젝트별 공유 디렉터리
- **문서 버전 관리**: 파일 변경 이력 추적
- **WebDAV API**: 외부 클라이언트(Windows Explorer, macOS Finder) 연동

### 1.2 시나리오상 역할: 문서 탈취 및 악성 파일 유포 거점

> **이 자산은 STEP 4-3에서 군 기밀 문서를 대량으로 탈취하고, 악성 파일을 유포하는 거점이다.**

공격자는 VPN 계정(GOV20190847)과 동일한 자격증명으로 Nextcloud에 로그인한다. 이후 WebDAV API를 통해 군 관련 문서를 대량 다운로드하고, 악성 파일을 공유 폴더에 업로드하여 내부 사용자 감염을 시도한다.

**취약점 요약:**

| 취약점 | 설명 | 심각도 |
|--------|------|--------|
| 크리덴셜 재사용 | VPN과 동일한 계정/비밀번호 | Critical |
| IP 접근 제한 없음 | 어떤 IP에서든 로그인 가능 | High |
| WebDAV 대량 다운로드 | 속도/건수 제한 없음 | High |
| 파일 업로드 제한 미흡 | 실행 파일(.exe, .bat 등) 업로드 가능 | Medium |
| 감사 로깅 미흡 | 대량 다운로드 패턴 알림 미설정 | Medium |
